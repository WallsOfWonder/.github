# 🎨 Welcome to WoW Labs 

**Building the next generation of Pay-What-You-Want creator tools and decentralized galleries on Stellar.**

At Walls of Wonder (WoW) Labs, we believe art should be universally accessible, and creators should be fairly compensated without losing revenue to intermediary fees. We are an open-source innovation lab building decentralized applications (dApps) and smart contracts to power zero-fee, on-chain artist patronage.

## 🚀 Our Mission
To bridge the gap between accessible home decor and fair artist compensation using the speed and efficiency of the Stellar network and Soroban smart contracts.

## 🛠️ Our Tech Stack
We are building a robust, modern ecosystem. If you love these technologies, you belong here:
* **Frontend:** React, Next.js, Tailwind CSS
* **Smart Contracts:** Rust, WebAssembly (Wasm)
* **Blockchain:** Stellar, Soroban, Freighter Wallet Integration
* **Storage:** IPFS for decentralized high-res art hosting
